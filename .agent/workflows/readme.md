# Workflow: Gerar/Atualizar README.md

Este workflow guia a criação ou atualização do arquivo README.md principal do projeto.

## Passos:

1. **Análise da Estrutura**:
   - Leia a estrutura de diretórios do projeto para identificar os componentes principais.

2. **Identificação Tecnológica**:
   - Analise `package.json`, arquivos de configuração e código para identificar as tecnologias usadas (React, Vite, TypeScript, etc.).

3. **Geração do Conteúdo**:
   - **Título e Descrição**: Nome do projeto e objetivo claro.
   - **Instalação**: Comandos necessários para rodar o projeto localmente (ex: `npm install`, `npm run dev`).
   - **Uso**: Exemplos de código ou fluxos de uso principais.
   - **Estrutura**: Representação em árvore (tree view) das pastas mais importantes.
   - **Variáveis de Ambiente**: Listar as chaves necessárias no arquivo `.env`.
   - **Deploy**: Instruções sobre como realizar o deploy para produção.

4. **Diretrizes de Tom**:
   - Profissional, conciso e direto ao ponto.
   - Utilize Badges quando apropriado para status, licença, etc.
