# Release Notes - v1.5-fiord-elegance-sinc

## 🚀 Sincronização Visual de Identidade

Esta versão sincroniza o aplicativo **Mobile** com a nova identidade visual **Fiord Elegance** do sistema administrativo, garantindo uma experiência de marca consistente em todas as plataformas.

### 🎨 Paleta de Cores Fiord
- **Cor Primária**: Restaurada para o Azul Fiord (`#47536f`), trazendo sobriedade e elegância.
- **Cor de Destaque**: Rosa Tapestry (`#b05d75`) para botões e elementos de ação.
- **Background**: Cinza Warm (`#f5f5f4`) para maior conforto visual.

### 📋 Melhorias na UI
- **Dashboard Mobile**: O card de "Admitidos" agora utiliza o gradiente da marca Fiord, melhorando o contraste e a hierarquia visual.
- **Elementos de Ação**: Botões e inputs foram atualizados para refletir os novos tokens de design.

## 🛠️ Alterações Técnicas
- `src/arquitetura/brand.config.ts`: Atualização dos tokens de cores globais.
- `src/arquitetura/index.css`: Sincronização das variáveis CSS e cores de tema do Tailwind.
- `src/App.tsx`: Ajustes finos em gradientes e estados visuais fixos.
